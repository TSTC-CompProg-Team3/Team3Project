User Permissions

Administration, Officers, and Teachers
- Users under this access level will be able to generate, submit, and edit attendance data for selected classes as well as print attendance reports for selected students.
Students and Parents
- Users under this access level can only view attendance data for the relevant student, users can also print attendance reports for the selected class.

Buttons

Back
- When clicked the program will take the user back to the main menu.
Edit
- When clicked the program will take the user to the edit attendance form where they can edit the specified date's attendance.
Submit
- When clicked the program will save the current data for attendance for the specified date. Changes made on the Attendance form can only be applied to the current day's attendance data, changes made to all other day's must be made from the Edit window.

Selectors

Attendance Date
- The user can use the attendance date selector to select a specific date to edit or view.
Student Selection
- The user can use the drop down box in the Edit form to select a specific student to view all of their attendance data for the current class.

Attendance Reports

Report
- Selecting the Attendance Report from the menu will generate a printable report of the relevant student's attendance for the selected class. Administration, Officers, and Teachers will need to select a student from the Edit form to generate a report for the selected student.
