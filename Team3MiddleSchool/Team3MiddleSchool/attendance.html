Buttons

Back
- When clicked the program will take the user back to the main menu.
Edit
- When clicked the program will take the user to the edit attendance form where they can edit the specified date's attendance.
Submit
- When clicked the program will save the current data for attendance for the specified date.

Date Selectors

Attendance Date
- The user can use the attendance date selector to select a specific date to edit or view.
