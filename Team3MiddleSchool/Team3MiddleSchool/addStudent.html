Buttons

Clear
- When clicked the program will clear all textboxes and calendar menus in the current form.
Submit
- When clicked the program will gather all of the information from the text boxes and calendar menus and add them to the database to create a new student.
Back
- When clicked the program will send the user back to the Admin/Officer menu.

Text Boxes

First Name
- The First Name text box is where the user will type a valid first name. This textbox will not allow numbers or special characters.
Last Name
- The Last Name text box is where the user will type a valid first name. This textbox will not allow numbers or special characters.
Email
- The Email text box is where the user will type a valid email. This textbox will check if the email is valid before it can be submitted.
Username
- The Username text box is where the user will type a valid email. This textbox will check if the username is null or empty, contains whitespace, has characters other than letters, digits, underscore and hyphen, and if it's length is between 3 and 20 characters. If the criteria is met then the program will upload the data to the database otherwise the user will be presented with a warning.
Password
- The Password text box is where the user will type a valid password. This textbox will check if the password is null or empty, contains whitespace, has atleast one character that is uppercase, and one number. It will also check to see if the length is between 8 and 20 characters. If the criteria is met then the program will upload the data to the database otherwise the user will be presented with a warning.
Phone Number
- The Phone Number textbox is where the user will enter a valid phone number. This textbox will check if the phone number is valid and if so it will then be able to be uploaded to the database. If the phone number is not valid then the user will be presented with a warning.
