Buttons

Clear
- When clicked the program will clear the textbox from the current form.
Submit
- When clicked the program will gather all of the information from the text box and add it to the database to create a new subject.
Back
- When clicked the program will send the user back to the Admin/Officer menu.

Text Boxes

Subject Name
- The Subject Name text box is where the user will type a new subject name that has not been used. If the subject name has been used then the user will be presented with a warning, if not then the information can be successfully uploaded to the database.
