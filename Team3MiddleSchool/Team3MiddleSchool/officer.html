Buttons

Add Course Button
- When clicked the program will open the add course form.
Edit Course Button
- When clicked the program will open the edit course form and populate the fields with the current information from the selected course.
Add Teacher Button
- When clicked the program will open the add teacher form.
Edit Teacher Button
- When clicked the program will open the edit teacher form and populate the fields with the current information from the selected teacher.
Add Subject Button
- When clicked the program will open the add subject form.
Edit Subject Button
- When clicked the program will open the edit subject form and populate the fields with the current information from the selected subject.
Add Student Button
- When clicked the program will open the add student form.
Edit Student Button
- When clicked the program will open the edit student form and populate the fields with the current information from the selected student.
Add Parent Button
- When clicked the program will open the add parent form.
Edit Parent Button
- When clicked the program will open the edit parent form and populate the fields with the current information from the selected parent.
Back
- When clicked the program will take the user back to the main menu.

Hidden Buttons

Remove Course Button
- This button will only be visible if the course is not currently tied to another table. When clicked the selected course will be deleted.
Remove Teacher Button
- This button will only be visible if the teacher is not currently tied to another table. When clicked the selected teacher will be deleted.
Remove Subject Button
- This button will only be visible if the subject is not currently tied to another table. When clicked the selected subject will be deleted.
Remove Student Button
- This button will only be visible if the student is not currently tied to another table. When clicked the selected student will be deleted.

Drop-down Menus

Course Select
- This dropdown menu allows the user to select the course they would like to either edit or delete.
Teacher Select
- This dropdown menu allows the user to select the teacher they would like to either edit or delete.
Subject Select
- This dropdown menu allows the user to select the subject they would like to either edit or delete.
Student Select
- This dropdown menu allows the user to select the student they would like to either edit or delete.
Parent Select
- This dropdown menu allows the user to select the parent they would like to edit.
