Buttons

Reset Button
- When clicked the assign classes form will reset to the default state with all Drop-down menus empty.
Edit Courses Button
- When clicked the student currently selected in the listbox will be assigned the classes that are currently selected in the Drop-down menus.
Back Button
- When clicked the program will take the user back to the main menu.

Student Select Box
- When a student is clicked the Drop-down menus on the form will populate with the current classes that student is assigned.
