Buttons

Grade Book
- When clicked this button will take the user to the Grade Book form with the corresponding class selected in the dropdown menu.
Attendance
- When clicked this button will take the user to the Attendance form with the corresponding class selected in the dropdown menu.
Seating Chart
- When clicked this button will take the user to the Seating Chart form with the corresponding class selected in the dropdown menu.
Return to Login
- When clicked this button will take the user back to the Login form.

Hidden Buttons

Officer Menu
- When clicked this button will take the user to the Officer Menu form that is only accessible to Officers.
Admin Menu
- When clicked this button will take the user to the Admin Menu form that is only accessible to Admins.
Assign Classes
- When clicked this button will take the user to the Assign Classes form that is only accessible to Admins and Officers.

Drop-down Menus

Class Select
- Using this dropdown menu the user can select which class they would like to view specific forms for. Only specific classes will be available for certain users.
