Buttons

Send Code
- When clicked if the email is valid and currently stored in the database then a password reset code will be generated and sent to the email.
Confirm
- When clicked if the code is valid then the program will check if the new password and the confirmation are the same and then update the password if valid.
Cancel
- When clicked the user will be taken back to the login page.

Text Fields

Email
- The email text field is where the user will type the email associated with their account.
Enter Code
- The enter code text field is where the user will type the code they received in their email after having the code sent.
New Password
- The new password text field is where the user will type in their new password to log into the program.
Confirm Password
- The confirm password text box is where the user will retype their new password to verify.
