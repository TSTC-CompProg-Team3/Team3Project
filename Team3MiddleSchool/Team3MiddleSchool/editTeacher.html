Buttons

Clear
- When clicked the program will clear all textboxes and dropdown menus in the current form.
Submit
- When clicked the program will gather all of the information from the text boxes and drop-down menus and add them to the database to edit the currently selected teacher.
Back
- When clicked the program will send the user back to the Admin/Officer menu.

Text Boxes

First Name
- The First Name of the currently selected teacher will be populated in the text box, and the user can chose to edit or keep the original first name. This textbox will not allow numbers or special characters.
Last Name
- The Last Name of the currently selected teacher will be populated in the text box, and the user can chose to edit or keep the original last name. This textbox will not allow numbers or special characters.
Email
- The Email of the currently selected teacher will be populated in the text box, and the user can chose to edit or keep the original email. This textbox will check if the email is valid before it can be submitted.
Username
- The Username of the currently selected teacher will be populated in the text box, and the user can chose to edit or keep the original username. This textbox will check if the username is null or empty, contains whitespace, has characters other than letters, digits, underscore and hyphen, and if it's length is between 3 and 20 characters. If the criteria is met then the program will upload the data to the database otherwise the user will be presented with a warning.
Password
- The Password of the currently selected teacher will be populated in the text box, and the user can chose to edit or keep the original password. This textbox will check if the password is null or empty, contains whitespace, has atleast one character that is uppercase, and one number. It will also check to see if the length is between 8 and 20 characters. If the criteria is met then the program will upload the data to the database otherwise the user will be presented with a warning.
