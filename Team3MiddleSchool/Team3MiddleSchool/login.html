Buttons

Login Button
- When clicked will verify users credentials and take them to the menu form.
Clear Button
- When clicked will clear all text fields in the login form.
Forgot My Password
- When clicked will bring the user to the forgot my password form where they can reset their password.

Text Fields

Username
- Where the user types in their username for the program.
Password
- Where the user types in their password for the program.

Checkboxes

Show Password
- When clicked the password field will reveal what the user has typed.
