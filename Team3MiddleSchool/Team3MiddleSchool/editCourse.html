Buttons

Clear
- When clicked the program will clear all textboxes and dropdown menus in the current form.
Submit
- When clicked the program will gather all of the information from the text boxes and drop-down menus and add them to the database to edit the current course.
Back
- When clicked the program will send the user back to the Admin/Officer menu.

Drop-down Menus

Teacher ID
- The Teacher ID drop-down menu will automatically be populated with the current selection for the course that is being edited. The user can then decide if they would like to change the teacher ID by making a different selection.
Subject ID
- The Subject ID drop-down menu will automatically be populated with the current selection for the course that is being edited. The user can then decide if they would like to change the subject ID by making a different selection.
Class Size
- The Class Size drop-down menu will automatically be populated with the current selection for the course that is being edited. The user can then decide if they would like to change the class size by making a different selection.

Text boxes

Class Name
- The Class Name text box will automatically be populated with the current selection for the course that is being edited. This textbox is where the user will type a new class name or keep the existing one.
