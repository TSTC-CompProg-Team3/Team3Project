Buttons

Main Menu
- When clicked the program will take the user back to the main menu.
Clear
- When clicked the program will clear any seats that are currently assigned to a student.
Random
- When clicked the program will generate a random seat assignment for each student in the current class.

Drop-down Menus

Seat Selectors
- Depending on the size of the current class there will either be 5, 10, or 20 available seats with seat selectors. Using the drop-down menus the user can manually assign a student to a seat or click the random button to randomly assign.

List Box

Available Students
- The available students list box contains the currently available students for the specified class.
