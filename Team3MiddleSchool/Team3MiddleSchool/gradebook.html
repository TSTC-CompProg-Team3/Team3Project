Buttons

Back
- When clicked the program will take the user back to the main menu.
Midterm Grades
- When clicked the program presents the user with a report for the midterm grades for the student the user is currently viewing.
Previous
- When clicked the program will cycle to backwards through the list of students grades.
Next
- When clicked the program will cycle to forwards through the list of students grades.
Remove
- When clicked the program will delete the currently selected grade from the gradebook table.
Edit
- When clicked the program will open a form that allows the user to edit the currently select grade in the gradebook table.
Add
- When clicked the program will open a form to add a grade into the gradebook for the specified student.

Text Boxes

Name
- The name text box displays the current student the user is viewing.
Class Participation
- The class participation text box displays the current total grade for class participation for the specified student.
Homework
- The homework text box displays the current total grade for homework for the specified student.
Quiz
- The quiz text box displays the current total grade for quizzes for the specified student.
Lab
- The lab text box displays the current total grade for labs for the specified student.
Test
- The test text box displays the current total grade for tests for the specified student.
Final
- The test text box displays the current total grade for the final for the specified student.
Total Grade
- The total grade text box displays the current total grade for all assignments for the specified student.

Tables

Grades
- The grades table displays a list of all current grades for the specified student.
